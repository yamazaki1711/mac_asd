# id-prosto.ru — Полный каталог ресурсов

## Структура сайта

### Публичные разделы (бесплатно, без регистрации)
- `/lists/{slug}` — Перечни ИД по видам работ (31 вид)
- `/forms/{nd-slug}/{form-slug}` — Формы ИД (бланки) по НД
- `/examples/{nd-slug}/{example-slug}` — Примеры заполнения ИД
- `/faq` — Вопросы и ответы

### Авторизованные разделы (платная подписка)
- `/projects/{id}/asbuiltdoc` — Комплекты ИД
- `/projects/{id}/gwl` — Общий журнал работ (6 разделов)
- `/projects/{id}/incominginspection` — Входной контроль
- `/projects/{id}/logbooks` — Специальные журналы (16 типов)
- `/projects/{id}/permitdoc` — Разрешительная документация
- `/projects/{id}/workdoc` — Рабочая документация

## Логика комплектов ИД (из приложения)

Комплект = {Объект + Раздел РД}

Структура комплекта:
1. Реестр ИД (автогенерация)
2. Акты освидетельствования (из ОЖР)
3. Журналы работ (специальные, автогенерация из записей)
4. Исполнительные схемы
5. Сертификаты и паспорта на материалы
6. Результаты испытаний

Специальные журналы (16 типов):
- anticorrosiveLogbook — Журнал антикоррозийной защиты
- boltsLogbook — Журнал болтовых соединений
- cablesLogbook — Журнал прокладки кабелей
- calibrationLogbook — Журнал тарировки ключей
- concreteWorksLogbook — Журнал бетонных работ
- couplingsLogbook — Журнал муфт
- designerInspectionLogbook — Журнал авторского надзора
- incomingInspectionLogbook — Журнал входного контроля
- pilesLogbook — Журнал свайных работ
- pipelinesLogbook — Журнал трубопроводов
- sheetsLogbook — Журнал замоноличивания
- steelStructuresLogbook — Журнал стальных конструкций
- weldingLogbook — Журнал сварочных работ
