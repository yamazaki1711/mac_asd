# Комплекты исполнительной документации по видам работ

> Источники: id-prosto.ru (31 вид работ, 39 НД), 
> Пособие по ИД Выпуск №2 (Сарвартдинова, 2026, 424 с.)

---

## Обязательный набор для ЛЮБОГО вида работ

1. **Разрешительная документация** (до начала работ):
   - Разрешение на строительство
   - Строительный паспорт / ГПЗУ
   - Проектная документация (прошла экспертизу)
   - Рабочая документация (с штампами «В производство работ»)
   - Приказы о назначении ответственных лиц
   - Свидетельства СРО

2. **Общий журнал работ** (Приказ 1026/пр):
   - Раздел 1: Сведения о лицах
   - Раздел 2: Журналы работ
   - Раздел 3: Документы
   - Раздел 4: Контроль
   - Раздел 5: Замечания
   - Раздел 6: Завершение

3. **Журнал входного контроля** (СП 543.1325800.2024 или ГОСТ 24297-2013)

4. **Акты освидетельствования** (Приказ 344/пр):
   - АОСР — скрытых работ
   - АООК — ответственных конструкций
   - АОУСИТО — участков сетей ИТО

5. **Исполнительные схемы и чертежи** (с фактическими размерами)

6. **Сертификаты/паспорта на материалы**

7. **Результаты испытаний** (лабораторных, и т.д.)

8. **Реестр ИД** (перечень всех документов комплекта)

## 1. Земляные работы (СП 45.13330.2017)

### Перечень ИД:
- АОСР: разработка выемок, устройство оснований, обратная засыпка
- Акт на скрытые работы: основания под фундаменты
- Журнал земляных работ (если предусмотрен проектом)
- Исполнительная геодезическая схема: котлован/траншея
- Результаты уплотнения грунта (лаб. испытания)
- Паспорта на грунт/щебень/песок
- Акт приёмки основания под фундаменты

### Ключевые формы:
- АОСР (Приказ 344/пр)
- Журнал входного контроля
- Протоколы испытаний уплотнения
