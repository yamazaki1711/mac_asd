# Qwen_ASD v11.0: Автономная Система ПТО, Смет и Права

Данный проект представляет собой локально-ориентированную (local-first) многоагентную систему, разработанную специально для **Mac Studio M4 Max**. Система предназначена для полной автоматизации работы департамента строительства (ПТО, Сметчики, Юристы).

## 🧠 Архитектура: "Железный Кулак" ASD
Система построена на базе **LangGraph**, что позволяет агентам работать нелинейно, обмениваться контекстом и возвращаться на предыдущие шаги для исправления ошибок.

### Основные Агенты:
*   👷 **Hermes (Project Manager)**: Главный оркестратор, принимает решения и координирует работу.
*   📐 **PTO Agent**: Анализ чертежей, извлечение ведомостей объемов работ (ВОР).
*   💰 **Smeta Agent**: Расчет стоимостей, работа с индексами и ФЕР.
*   ⚖️ **Legal Agent**: Поиск юридических «ловушек» в контрактах, проверка на соответствие ФЗ-44/223.
*   🔄 **Reflection Node**: Модуль самообучения, обучающий систему на основе аудита выполненных задач.

## 🛠 Технологический Стек
*   **Orchestration**: LangGraph, StateGraph.
*   **LLM**: Ollama (Gemma 4 31B Cloud / Local).
*   **Knowledge Base**: Obsidian (Wiki-backed rules & skills).
*   **Databases**: 
    *   **PostgreSQL + pgvector**: Векторная память (RAG).
    *   **Neo4j**: Граф связей тендеров и документов.
    *   **Redis**: Кэш состояний.

## 🚀 Быстрый Старт
1.  **Инфраструктура**:
    ```bash
    cd infrastructure
    docker compose up -d
    ```
2.  **Запуск системы**:
    ```bash
    source .venv/bin/activate
    export PYTHONPATH=$PYTHONPATH:.
    python src/main.py
    ```

## 📈 Функция Самообучения
Система автоматически анализирует логи своей работы и обновляет инструкции в папке `data/wiki/`. Это позволяет ИИ становиться умнее с каждым новым тендером, не меняя основной код.
